1. Conexión con GitHub

Necesario para publicar. El token queda guardado solo en este navegador.

Creá uno en GitHub → Settings → Developer settings → Fine-grained tokens, dándole acceso solo a este repositorio.

2. Nuevo producto

Completá los datos. Los campos con * son obligatorios.

3. Fotos

Subí una o varias. La primera es la principal. Se guardan en assets/<marca>/.

Arrastrá las fotos acá o hacé click para elegir

4. Publicar

Sube las imágenes y agrega el producto al catálogo con un commit. El sitio se actualiza en ~1 minuto.
